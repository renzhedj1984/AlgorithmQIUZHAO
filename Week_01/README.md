# Week_01 学习笔记
## 如何高效学习
1.  重复观看培训课程，强化对知识点，加深记忆
2.  重复刷题，每个题目保证最少刷5遍，不死磕，敢于硬性记忆代码并理解
3.  Leetcode日常刷题，看高票&高质量题解
